# Planificador Semanal OAC

📅 Planificador Semanal OAC | Gestión Eficiente de Atención al Ciudadano

El Planificador Semanal OAC es una herramienta institucional de código abierto diseñada para la organización interna y la optimización de trámites. Enfocada en la privacidad y la soberanía tecnológica, permite programar tareas, proyectos y atención de casos mediante listas dinámicas y un calendario integrado.


🚀 Funciones Principales

Multiplataforma: Acceso desde cualquier equipo de la oficina (Windows, Linux, Web).

Modo Claro/Oscuro: Interfaz adaptable para largas jornadas de trabajo.

Listas de Tareas Personalizadas: Organización por tipo de trámite o departamento.

Arrastrar y Soltar: Reasignación rápida de prioridades diarias.

Sub-tareas y Notas: Desglose detallado de expedientes complejos.

Soporte Markdown: Documentación con formato profesional.

Almacenamiento Local/Seguro: Garantiza la protección de datos del ciudadano.

Colores por Categoría: Identificación visual de urgencias (Atención Directa, Trámites, Terreno).

Recordatorios e Interfaz en Español: Diseñado para el flujo de trabajo público.


🛠️ Instalación y Ejecución (Desarrollo)

El sistema está construido sobre una arquitectura robusta que combina la agilidad de JavaScript y PHP con la seguridad de tipos de Hack.


Requisitos Previos:

Necesitarás tener instalado Git, Node.JS (v16.X recomendado), Composer y el entorno HHVM para el soporte de Hack.

Clonar el repositorio institucional:

Bash
git clone https://github.com/Abdoesntwork1205/Planificador-Semanal-OAC
cd PlanificadorOAC


nstalar dependencias:

Bash
yarn install
composer install


Ejecutar en entorno local:

Para la versión web: yarn run serve

Para la versión nativa (Desktop): yarn run electron:serve

Docker (Despliegue Rápido)

Para levantar una instancia de desarrollo rápida en la red interna:

Bash
docker-compose up


🏛️ Sobre el Proyecto

Este planificador es una iniciativa para modernizar la gestión pública, permitiendo que la OAC mantenga un registro transparente y organizado de sus compromisos semanales, mejorando así el tiempo de respuesta al ciudadano.
